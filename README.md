# SEAC Gestão - Fluxo de Caixa Agrícola

## Visão Geral

O SEAC Gestão é um sistema web para controle financeiro e operacional de propriedades rurais, focado em fluxo de caixa agrícola. Permite o cadastro e acompanhamento de receitas, despesas, áreas, propriedades, fornecedores e centros gerenciais, facilitando a tomada de decisão e a gestão eficiente do agronegócio.

## Funcionalidades

- **Dashboard Financeiro**: Visualização gráfica das movimentações, saldos bancários e indicadores financeiros.
- **Cadastro de Receitas e Despesas**: Controle detalhado de entradas e saídas, com categorias, formas de pagamento/recebimento, status e centro de custo.
- **Gestão de Propriedades e Áreas**: Cadastro de propriedades rurais, áreas produtivas e suas atividades.
- **Relatórios Personalizados**: Geração e exportação de relatórios em diversos formatos (Excel, PDF, CSV).
- **Gestão de Fornecedores**: Cadastro e consulta de fornecedores do agronegócio.
- **Centros Gerenciais**: Organização financeira por centro de custo.

## Tecnologias Utilizadas

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase (Banco de dados e autenticação)

## Como executar o projeto

1. Clone o repositório:
    ```sh
    git clone <YOUR_GIT_URL>
    cd seac-agri-dashboard
    ```
2. Instale as dependências:
    ```sh
    npm install
    ```
3. Inicie o servidor de desenvolvimento:
    ```sh
    npm run dev
    ```
